ABSTRACT:

Construção de um servidor que permite utilizar as operações CRUD num arquivo sonoro fornecido pelo docente da UC.

Para esse efeito foi criado um ficheiro JSON cancoes.json para ser o armazenamento do arquivo.

Para adicionar um id a cada música foi utilizado um nanoid na função definida no ficheiro JS adicionar.js

Deliverables:

  • Layout de todas as páginas criadas
  • Página inicial
  • Página de erro
  • Página de uma música em específico
  • Página usada para editar uma música
  • Router da aplicação
  • Base de dados fornecida
