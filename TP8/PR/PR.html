ABSTRACT:

Este trabalho de casa encontra-se divido em 2 exercícios:

No 1.º exercício era pedido para fazer a conversão do dataset disponibilizado em página para um outro formato. Foi utilizado o formato JSON para armazenar essa a informação, uma vez que se mostrou ser o formato mais útil. Este ficheiro foi posteriormente importado para o MongoDB

Já o 2.º exercício consistia na criação de um servidor que permitia gerir a base de dados em causa e, ainda, suportar uma determinada API de dados.
